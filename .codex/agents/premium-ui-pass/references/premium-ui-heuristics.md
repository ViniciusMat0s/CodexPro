# Premium UI Heuristics

Use these heuristics to evaluate whether an interface feels premium.

## Hierarchy
- Is the most important thing visually obvious?
- Do headings, supporting text, and actions have clear rank?
- Does the eye know where to go first?

## Spacing
- Is spacing deliberate and rhythmic?
- Is the UI breathing enough?
- Does any area feel cramped or randomly padded?

## Composition
- Does the page feel designed as a whole?
- Are sections grouped meaningfully?
- Is the layout balanced?

## Density
- Is the page dense enough to be useful but calm enough to feel premium?
- Are cards/panels overloaded?

## Action clarity
- Is the primary CTA obvious?
- Are destructive/secondary actions clearly differentiated?

## Quality perception
- Would this interface feel credible in a premium SaaS product?
- Or does it feel like a generic starter template?
