# Token and Primitive Guidelines

## Tokens
Use shared tokens or conventions for:
- spacing
- typography
- radii
- shadows
- semantic colors

## Primitives
Prefer primitives for:
- button
- input
- textarea
- select
- card
- badge
- table shells
- modal/sheet/dialog shells

## Avoid
- arbitrary one-off styling everywhere
- duplicate primitives with slight naming differences
- inconsistent visual behavior for equivalent actions
