# Frontend Review Checklist

## UI quality
- Strong hierarchy
- Clean spacing
- Polished cards/forms/tables
- Responsive quality
- States handled well

## UX
- Primary action obvious
- Labels clear
- Feedback after actions clear
- Flow understandable

## Accessibility
- semantic structure
- focus visibility
- form labels
- button/link correctness

## Engineering
- component boundaries make sense
- no giant overstuffed components
- no repeated markup that should be reused
- no unnecessary client-component spread
