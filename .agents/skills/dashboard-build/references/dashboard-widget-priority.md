# Dashboard Widget Priority

## High priority widgets
- KPIs tied to decisions
- urgent statuses
- alerts requiring action
- high-signal summaries

## Medium priority widgets
- trends
- comparisons
- recent activity
- grouped summaries

## Lower priority widgets
- secondary metadata
- tertiary charts
- nice-to-have summaries
- duplicated information

## Review questions
- Does each widget earn its space?
- Can any widget be merged or demoted?
- Is the first screen helping the user act?
