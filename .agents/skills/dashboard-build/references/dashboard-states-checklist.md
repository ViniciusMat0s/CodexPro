# Dashboard States Checklist

## Loading
- Structure remains recognizable
- Skeletons or placeholders feel intentional

## Empty
- Explains why no data exists
- Suggests what to do next

## Error
- Useful explanation
- Retry or recovery path where possible

## Partial data
- Missing panels handled gracefully
- Does not break composition

## Responsive
- KPI order still makes sense
- Tables degrade well
- Actions remain visible
