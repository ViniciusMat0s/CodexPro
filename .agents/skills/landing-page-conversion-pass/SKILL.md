# Purpose

Use this skill for landing pages that need to convert better and feel more premium.

This skill is for:
- hero optimization
- section order refinement
- CTA hierarchy
- trust signal placement
- premium visual composition
- reducing clutter and friction
- improving scanability and message flow

# Use this skill when

Use this skill when the request involves:
- improving a landing page
- increasing clarity and conversion readiness
- making the page feel premium instead of generic
- refining hero, benefits, proof, CTA, and section structure
- optimizing a SaaS, service, or product sales page

# Do not use this skill when

Do not use this skill for:
- dashboard/product UI
- admin surfaces
- auth flows
- generic visual tweaks with no conversion significance

# Conversion philosophy

A good landing page should answer quickly:
- what this is
- who it is for
- why it matters
- why it is credible
- what to do next

# Workflow

## 1. Evaluate the current conversion path

Inspect:
- hero clarity
- headline quality
- subheadline support
- CTA prominence
- section order
- proof/trust placement
- friction points
- clutter and message competition

## 2. Improve message hierarchy

Strengthen:
- value proposition clarity
- supporting explanation
- differentiation
- CTA visibility
- scanability
- repetition of core message without redundancy

## 3. Improve page composition

Prefer:
- deliberate section rhythm
- strong whitespace
- trust-building visuals
- clearer grouping
- reduced noise
- cleaner social proof and proof blocks
- more intentional CTA placement

## 4. Review conversion friction

Check:
- unclear offers
- weak CTA language
- excessive content above the fold
- generic features with no user value framing
- missing trust signals
- poor mobile flow

## 5. Final summary

Summarize:
- conversion issues improved
- messaging hierarchy changes
- CTA or trust changes
- page-flow improvements
- remaining weak spots if any

# Quality bar

The page should feel:
- premium
- clear
- credible
- conversion-aware
- visually intentional
- easy to scan
